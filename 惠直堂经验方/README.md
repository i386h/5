　　惠直堂经验方 清 陶承熹

　　序

　　昔仓公挟方术周行天下。历久而术益工。方益多。是知治病不可无方。而方尤不可不蓄之富庄按不可奇珍如是者济人。

　　方。分为备急方。共聚五。好义先大夫集之忧。今孙子耶。虽然。

　　方凑病。苟若无方矣。

　　可。
