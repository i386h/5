　　钦定四库全书　　　　子部五
　　提要
　　济生方八卷　　　　　医家类
　　【臣】等谨案济生方八卷宋严用和撰用和始末未详考吴澄集有易简归一方序称严子礼剽陈氏三因方之论而附以经騐之药以其名推之子礼似即用和字其人盖在陈言后矣澄又有古今通变仁夀方序曰世之医师不一惟有所传授得之尝试者多騐予最喜严氏济生方之药不泛不繁用之輙有功盖严师扵刘其方乃平日所尝试而騐者也则澄盖甚重此书矣其书分门别类条例甚备皆立论于前而以所处诸方次列于后自序称论治凡八十制方凡四百总为十卷用之十五年收效甚多因锓梓以传是当时尝板行扵世后经亡佚医家但辗转援引如归脾汤至今尚冠以济生之名盖其遗法今检永乐大典中所采尚多以类裒辑得论五十六方二百四十余厘为八卷虽不能及原书之数然其所阙亦仅十之三四而已书中议论平正条分缕析往往深中肯綮如论补益云药惟平补柔而不僣专而不杂间有药用羣队必使刚柔相济佐使合宜又云用药在乎穏重论欬云今人治喜用伤脾之剂服之未见其效谷气先有所损论吐衂云寒凉之剂不宜过进诸方备列参而用之盖其用药主扵小心畏慎虽不善学之亦可以摸棱贻悮然用意谨严固可与张从正刘完素诸家互相调剂云乾隆四十六年九月恭校上
　　总纂官【臣】纪昀【臣】陆锡熊【臣】孙士毅
　　总　校　官【臣】陆　费　墀
　　原序
　　余夙嗜名书蚤印师授以医道行世五十余年此因暇日论治凡八十制方凡四百总为十卷号济生方出而用之十有五年收效甚多用锓诸梓以广其传或谓古者处剂不过数种针炙不过数处君之方奚以多为余应之曰医者意也生意在天地间一息不可间断续此方所以续此意续此意所以续此生请勿以多议余严用和序








　　钦定四库全书
　　济生方卷一　　　　　　宋　严用和　编制方
　　论曰一隂一阳之谓道偏隂偏阳之谓疾夫人一身不外乎隂阳气血相与流通焉耳如隂阳得其平则疾不生隂阳偏胜则为痼冷积热之患矣所谓痼冷者隂毒沈痼而不解也积热者阳毒蕴积而不散也故隂偏胜则偏而为痼冷阳偏胜则偏而为积热古贤云偏胜则有偏害偏害则为偏絶不可不察也大扺真阳既弱胃气不温复啖生冷氷雪以益其寒隂沍于内阳不能胜遂致呕吐涎沫畏冷憎寒手足厥逆饮食不化大腑洞泄小便频数此皆隂偏胜而为痼冷之证也其或隂血既衰三焦已燥复饵酒炙丹石以助其热阳炽于内隂不能制遂致口苦咽干涎稠目澁膈热口疮心烦喜冷大便闭结小便赤淋此皆阳偏胜而为积热之证也施治之法冷者热之热者冷之痼者解之积者散之使隂阳各得其平则二者无偏胜之患矣
　　补益
　　论曰夫人禀中和之气以生常能保守真元何患乎有病焉不善卫生者思虑役其智嗜欲乱其真荣卫一虚因兹积防成损积损成衰及其病也既不能御气以全身又不能饵药以延夀圣人有言曰治未病不治已病治未乱不治已乱夫病已成而后药乱已成而后治不亦晚乎凡人有虚损之病及早为之补益庶有延龄之望后方所载药性平补柔而不僭専而不杂间有药用羣队必使刚柔相济佐使合宜可以取效前贤之书有单服附子之戒者正虑其肾恶燥也既欲用一刚剂专而易效须当用一柔剂以制其刚庶几刚柔相济不特取效之速亦可使无后患后方所载参附沈附茸附耆附是矣若阳虚隂极之证当用姜附三建诸丹之类或升或沈为之佐使欲其刚多于柔不致太僭庶免炎上之患用药在乎稳重故也
　　补真丸　凡不进饮食以脾胃之药治之多不效者亦有谓焉人之有生不善摄养房劳过度真阳衰虚坎火不温不能上蒸脾土冲和失布中州不运是致饮食不进胸膈痞塞或不食而胀满或已食而不消大腑溏泄此皆真火衰虚不能蒸蕴脾土而然古人云补肾不如补脾余谓补脾不如补肾肾气若壮丹田火经上蒸脾土脾土温和中焦自治膈开能食矣
