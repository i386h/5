　　木香饼

　　治一切气滞结肿，或痛或闪，及风寒所伤作痛，并效。

　　[卷之十四　乳病门] 乳悬

　　芎归汤

　　治产后瘀血上攻，忽两乳伸长细小如肠，直过小腹，痛不可忍，名曰乳悬，危证。

　　川芎当归（各一斤）上用水煎浓汤，不时温服，再用二斤，逐旋烧烟安在病患面前桌子下，令病患曲身低头，将口鼻及病乳常吸烟气，未甚缩，再用一料，则瘀血消而乳头自复矣，若更不复旧，用蓖麻子捣烂贴顶上，片时收，即洗去。

　　[附] 保生碎事

　　（济阴慈幼外编）

　　漪子

　　曰：尝观《列女传》曰：古之妇人妊子，坐立寝食，视听言动，皆要循循合矩，如太任之性，端一诚庄，惟德之行，及娠文王，未尝不以胎教为重，生子卒为周宗，其理可不验与。今之孕妇，未诺古法，为夫者当以胎教之理谕之，俾妇善调固其本，以达其支，非惟无堕失之患，抑且生子形容端正，才器过人矣。

　　又曰：人禀阴阳之气以生，借父精母血以成其形。人始胎，一月如露珠，二月似桃花，三月男女分，四月形象具，五月五脏生，六月六腑成，七月关窍通，八月游其魂，九月三转身，十月受气足，始生。所以古人云：有孕不在房，养子痘疹稀。以慎重其胎也，世人可不谨欤。

　　今幼科医书多种，自有专门，余集只录堕地时至七日内之事，采其最要者，简而当，详而明，有先后原委，切要着明，其良法实实可防患于未然耳。朽淇年已逾甲，志在守庚，杜却家缘，养山寺，向梓证道并地理碎事诸刻，以求政四方，此亦不过幼吾幼之婆心焉耳。

　　[附　保婴经验方] 梦生方

　　初生不能发声，谓之梦生，切不可断脐，以绵衣包儿，用大油纸捻烧脐，待气回方可断也，将胞衣连带用火炙，令暖气入儿腹，取一猫，用青布裹头目，令一伶俐女人拿住猫头，向儿耳旁，以口咬破猫耳，猫大叫一声，儿即醒矣。

　　初生无啼，是肛门为脂皮所塞，用簪通开。

　　外台秘要

　　云：儿生不啼，由产难少气也，取脐带向身却捋之，令气入腹，仍呵至百度，自啼矣。又以葱白徐徐鞭之，亦妙。

　　菖蒲捣生汁，入口即活。

　　生儿啼声不出及声不转者，舌下有膜如石榴子，或口中上有泡，速以指爪摘断之，用苇刀于割之亦可，微有血出即活，若血多者，烧断发同猪脂涂之。

　　小儿因产难，产下不哭，微有气者，以本父母真气度之，即活。

　　[附　保婴经验方] 拭口法

　　婴儿在胎，口有热物，盖胎毒也，生下啼声未出，急用软帛或绵裹指拭尽，贵在神速，迟则咽下，伏之于心，遇天行时气，久热不除，乃乘于心，心主血脉，得热而散，流溢于胃，胃主肌肉，发出于外，成疮疹之毒，世之小儿无可免者，不知病源在此，若根据此法，庶可免矣。

　　产家要诀

　　云：予见有挖出血饼，切开色成五彩，举家大喜，谓必无痘患矣，而竟以患痘死者。且口血虽咽，必入肠胃，大便一通，其血亦去，岂有径投极下之处，伏于枣大有肾之中，藏蓄不动，久而后发者哉。太医院支改斋先生辨之甚悉，谓痘之所主，皆属心火。斯言足破千古疑案，及思古方用黄连甘草抹儿口以解胎毒者，岂非泻心之药也乎。

　　[附　保婴经验方] 甘草法

　　凡妊娠临月，预办好甘草指许一节拍碎，以水二蚬壳煎一蚬壳，以绵裹蘸，令儿吮之，若吐出恶物为佳，服一蚬壳不吐，即不须服，不问小儿虚实，皆可服之。

　　[附　保婴经验方] 黄连法

　　临产时，先以黄连五钱，甘草二钱五分，熬浓汁，生下即与二三匙，再加朱砂细末调汁，抹儿口中，打尽腹中旧屎，次用朱蜜。

　　[附　保婴经验方] 朱蜜法

　　葛氏肘后方

　　云：先用甘草、黄连拭口，吐其恶汁，稍定更与朱蜜，安神定魄，用好朱砂一大豆许细研，水飞过，炼蜜一蚬壳许，和朱砂成膏，旋抹口中，非独镇心定神，亦解恶毒，若面色青白，啼声不响，即不须服。

　　[附　保婴经验方] 断脐法

　　产家要诀

　　云：儿出胎洗后，方断脐带，则不伤水生病。断脐须捋汁令尽，否则寒湿入腹，或作脐风，又须于近脐四五寸处以线紧扎，以帛包裹，以口咬断。盖扎紧则儿血不贯于胞衣，自然痿缩弗胀而易下，即或延缓数日，亦无大害。口咬则断脐不犯乎刀剪，自无冷气内侵，可免腹中吊痛之虞。如或天时寒冱，坐草艰难，母子劳伤元气者，先扎脐带，以油纸捻点火烧断，令暖气入儿腹中，此又为回生起死之妙法。近用蕲艾灸脐亦妙。所留脐带不可太长，太长则难干而伤肌，且恐引外风为脐风，亦不可太短，太短则逼内而伤脏，或致成腹痛而夜啼，只以儿之足掌长短为度，乃酌中之法。《千金论》云：须令至儿足趺上为度。

　　造道集

　　云：儿始生宜净洗，则燥血不留于折路之间，可令皮肤光泽，然后剪脐。脐乃初生之蒂也，剪之宜长一尺有二，用粗线缚紧，剪不长，多生脐风，缚不紧，阴间虚肿。

　　缪仲淳广笔记

　　云：儿初生不可剪脐带，三朝用面和水成薄饼，置儿腹，穿脐带于面上，将蕲艾火灸脐带近脐处，或三炷，或五七炷，须下帐避风，灸毕，仍将脐带扎好，听其自脱，至七日方脱者，元气足也。

　　凡儿生下，每日夜将清汤或苦茶蘸软绢搅儿口内，如齿边有白点，即以指爪或细针挑破，取桑树内汁滤清涂之，永无惊风撮口之患。

　　[附　保婴经验方] 裹脐法

　　千金论

　　云：裹脐须令白练揉软，方四寸，新绵浓半寸，与帛等合之，缓急得中，急则令儿吐，不可轻解，解时闭户下帐，冬间用火，令帐内温暖，仍以温粉敷之，若脐不干，烧绛帛敷之。

　　枯矾五钱，龙骨一钱，黄丹一钱，麝香少许为末，敷之。

　　乱发烧灰，敷之亦可。或云黄柏末敷，妙。

　　[附　保婴经验方] 脐风方

　　张涣论

　　云：婴儿断脐后，宜用浓帛裹护，若乳母不慎，或因浴水入，或儿尿绷中未换，湿气所侵，或当风解视，七日内遂有此疾，令儿脐肿多啼，不能乳哺，即成脐风也。

　　脐肿，当随轻重，重者灸二三十次，轻者但汁出多啼，用当归末和胡粉敷之，常用热絮熨之。

　　初生七日内，面赤喘急，啼声不出，名曰嘬口脐风，齿龈上有小泡如粟米状，急以温水蘸青软帛或绵裹指，轻轻擦破，即开口，不须服药。

　　白僵蚕直者四枚炒，去丝嘴，为末，蜜调敷唇内，即愈。

　　缪仲淳

　　云：嘬口之症，必先大便热，须用生犀角及真羚羊角，磨和蜜汁饮之，有效，急则用大黄二钱甘草二钱煎服。

　　[附　保婴经验方] 口噤方

　　葱白一寸破四片，乳汁蒸灌之。

　　[附　保婴经验方] 浴儿方

　　北方生子多不洗浴，初生下但以旧绵拭净，直至三四岁出痘后方浴，或大小便处略以水揩抹之，故北方人较南方壮实，不徒风气然也。若毕竟欲浴，取益母草五两锉细，煎滚，温温浴之，不生疮疥，又可祛寒气风湿。

　　产家要诀

　　云：儿初出胎浴洗，汤内宜入食盐少许，洗毕拭干，以腻粉一钱许研细，摩其遍身及两胁，然后绷裹，既不畏风，又散诸气。今北方三朝方洗，将绷裹之儿又复解开，多致感冒惊风。盖洗三虽云古礼，而防患贵在变通，只根据前法，出胎便洗，自是稳当。

　　保生要方

　　云：小儿初生，候浴水不得，且以绵絮包裹，大人怀中暖之，浴后仍当如此，虽暑月绵絮亦宜渐渐去之，乍离母腹，未可冒寒气也。预煎沸汤，以甑贮之，临时热洗，不犯生水，则儿不生疮。

　　[附　保婴经验方] 乳儿方

　　保生要法

　　云：小儿初生，若多睡，勿强与乳，自然长而少病。

　　颅囟经

　　云：夏中盛热时，乳母浴后便与儿乳，使儿成胃毒赤白痢。定息良久，熟揉乳之可也。

　　千金论

　　云：乳儿不可过饱，满而溢则成呕吐，大饱，以空乳吮之即消，若乳来猛，取出再乳。凡初乳时，须常捏去宿乳，夏不去热乳，令儿呕逆，冬不去冷乳，令儿咳痢。母与儿卧，以臂枕之，令乳与头平，则儿不噎。母欲寐，即夺其乳，恐睡熟则不如饱足，而成呕吐也。

　　小儿精要

　　云：夜间乳儿，须乳母起坐，抱儿乳之。

　　母有娠乳儿，必患胎黄。母大醉乳儿，必患惊热。母伤饱乳儿，必患喘急。母新房乳儿，必患羸瘦。母大怒乳儿，必患癫狂。母新吐乳儿，必患虚羸。

　　琐碎录

　　云：儿睡，忌乳母鼻风吹囟门，成风疾。

　　[附　保婴经验方] 哺儿方

　　葛氏肘后方

　　云：儿生三日，应开腹，助谷神，壮胃气，用粟米煮熟，研如乳汁，与儿大豆许，慎不可与杂药。

　　乳者奶也，哺者食也，乳后不得与食，哺后不得与乳，乳食相并，难以克化，大则成癖，小则成积，疳气自此始矣。
